# hexdump
A simplified clone of unix hexdump utility.

## Compiling
```bash
make all
```
## Usage
```bash
kapil@DESKTOP-Q6HEFQA:~/repos/hexdump$ ./hexdump test.txt
00000000  54 68 69 73 20 69 73 20  74 65 73 74 20 66 69 6C  |This is test fil|
00000010  65 2E 0A                                          |e..|
00000013
```
